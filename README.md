# Language Learning App
This is a simple Language Learning App I created as part of my CodeAlpha Internship.
The main idea of this project is to help users learn new words in an easy way. It works like a flashcard system where you can see a word, try to remember the meaning, and then check the answer.

## What this app can do
- Shows words and meanings
- Next button to move through words
- Show Meaning button to check answers
- Speak option to hear pronunciation
- Add your own words
- Basic quiz to test yourself
- Words are stored, so they don’t disappear after refresh

## Technologies used
- HTML  
- CSS  
- JavaScript  

## How to use
Just open the index.html file in your browser.  
No installation is needed.

## About this project
I built this project to understand how user input, data storage, and simple logic work together in a web application. It also helped me practice JavaScript in a practical way.
