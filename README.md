# Language Learning App

This project is a simple Language Learning App developed as part of my CodeAlpha Internship.

## About the Project
This application helps users learn new words and their meanings. Users can navigate through words, view meanings, and add their own words.

## Features
- View words one by one
- Show meaning of each word
- Add new words
- Data is saved using local storage

## Technologies Used
HTML, CSS, JavaScript

## How to Run
Open index.html in your browser
